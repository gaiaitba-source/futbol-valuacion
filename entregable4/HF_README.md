# ⚽ Scout AI — Valuación de futbolistas a 12 meses

Estima el valor de mercado futuro de un jugador (con intervalo de confianza), muestra su
trayectoria real, la evolución del mercado, un ranking de oportunidades y un resumen de scouting
con IA. Entregable 4 — Ciencia de Datos Aplicada, ITBA.

La app corre la API (FastAPI) y la interfaz (Streamlit) en un único contenedor.
